Модели

В разделе рассмотрены вопросы работы с моделями системы: как непосредственно создавать, редактировать, копировать или удалять модели. Какие есть инструменты для работы с моделями и другие подобные вопросы.

Модель системы -

Добавление модели

Для добавления новой модели перейдите на страницу моделей, нажав в пункте меню "Модели"

Для того чтобы добавить новую модель, нажмите "Добавить модель"

Введите название модели и нажмите кнопку "Добавить"

Редактирование модели

После создания модели, Вас перенаправит на страницу редактирования модели. Для того, чтобы попасть в редактор моделей, нажмите название модели.

Страница редактирования моделей содержит область для отображения моделей, редактор с помощью которого можно добавлять новые узлы, связи, топологии и менеджеры управления, а также кнопки переходя назад и на страницу "Нагрузки"

Редактор позволяет добавлять, удалять, перемещать и изменять узлы связи

Режимы редакторов

  • Режим «Редактор узлов» позволяет добавлять новые узлы в зависимости от их типа, задавать значения параметрам, перемещать, удалять и редактировать их
  • Режим «Редактор топологии» позволяет задавать и редактировать попарные связи узлов
  • Режим «Редактор путей» позволяет задавать и редактировать маршруты передачи мультимедийных потоков от одного узла к другому
  • Режим «Редактор менеджеров управления» позволяет задать попарные связи узлов управления с другими узлами и редактировать их

Элементы управления

  • Для добавления нового узла или связи нажмите кнопку "Добавить".
  • Для изменения узла выделите узел правой кнопкой мыши и нажмите кнопку "Изменить". Для изменения связи или топологии, нажмите левой кнопкой мыши по первому узлу и правой кнопкой мыши по второму узлу и нажмите кнопку "Изменить"
  • Для удаления элемента выделите левой или правой кнопкой мыши по элементу и нажмите кнопку "Удалить".
  • Для перемещения элемента по карте, выделите правой кнопкой мыши по элементу и нажмите кнопку "Переместить, затем нажмите левой кнопкой мыши по области моделирования куда нужно переместить элемент, а затем еще раз нажмите кнопку "Переместить"
  • Внизу экрана располагается кнопка для скриншота модели (P) и кнопка для того, чтобы скрыть название узлов (Т)

Добавление и удаление узлов

  • Для добавления нового узла выберите "Редактор Узлов", затем нажмите кнопку добавить и заполните форму для добавления узла.

  1. Введите название узла
  2. Выберите тип узла
    • Ретранслятор дублируют сигнал и передают их далее клиентам или другим мультимедийным узлам.
    • Репликатор это источники мультимедийных потоков (также могут быть ретрансляторами)
    • IP-узлы это связующие звенья между мультимедийными узлами и характеризуют внешнюю нагрузку на систему

    • Узел-квадрат(репликатор) может создавать на основе входящего - новые потоки внутри себя и потом от них передавать в другие узлы
    • Узел-круг(ретранслятор) не могут создавать потоков, они только передают те потоки, которые в них вошли.

  3. Является ли узел менеджером управления
  4. Введите IP-адрес узла(каждый узел должен иметь уникальный IP-адрес)
  5. Укажите максимальную пропускную способность входящего и исходящего канала
  6. Введите максимальный объём памяти, объем ресурсов процессора и популярность узла
  7. Нажмите левой кнопкой мыши по нужной части области моделирования или введите координаты где будет распологаться узел
  8. После заполнения формы, нажмите кнопку "Добавить"
  9. Кнопка "Заполнить", заполняет форму данными по-умолчанию
  • Для удаления узла выделите правой кнопкой узел и нажмите кнопку "Удалить"

Добавление и удаление топологии

  • Для добавления новой топологии выберите "Редактор Топологии", затем выделите левой кнопкой мыши первый узел и правой кнопкой мыши второй узел и нажмите кнопку "Добавить" и заполните форму для добавления топологии.

  1. Введите ширину канала для связи
  2. Нажмите кнопку "Добавить"
  • Для удаления топологии выделите правой кнопкой первый узел и левой кнопкой мыши второй узел и нажмите кнопку "Удалить"

Добавление и удаление связей

Для добавления новой связи выберите "Редактор Путей", затем выделите левой кнопкой мыши первый узел и правой кнопкой мыши второй узел и нажмите кнопку "Добавить" и заполните форму для добавления связи.

  1. Введите название связи
  2. Выберите поток-родитель
    • Для добавления потока-родителя, нужно сделать связь внутри узла (связь внутри узла можно делать только для репликатора(круг-квадрат))
      1. Выделите левой кнопкой мышки первый узел и правой кнопкой мыши второй узел и нажмите кнопку "Добавить" связь
      2. Заполните информацию по связи
      3. Нажмите кнопку "Добавить"
  3. Укажите качество связи и занимаемую пропускную способность связи
  4. Нажмите кнопку "Добавить"
  • Для удаления топологии выделите правой кнопкой первый узел и левой кнопкой мыши второй узел и нажмите кнопку "Удалить"

Нагрузка

В разделе рассмотрены вопросы работы с нагрузкой: как непосредственно создавать, редактировать, копировать или удалять нагрузку. Какие есть инструменты для работы с нагрузками и другие подобные вопросы.

Нагрузка -

Добавление новой нагрузки

  • Для добавления новой нагрузки нажмите кнопку "Добавить нагрузку"

  1. Выберите модель для которой нужно добавить нагрузку
  2. Введите название
  3. Укажите количество шагов
  4. Выберите алгоритм работы при перегрузке
  5. Выберите как будет моделироваться статистика
  6. Нажмите кнопку "Добавить"

Редактирование нагрузки

После создания модели, Вас перенаправит на страницу редактирования модели. Для того, чтобы попасть в редактор моделей, нажмите название нагрузки.

Страница редактирования нагрузки содержит область для отображения моделей, редактор с помощью которого можно добавлять новую нагрузку на узел,а также кнопки перехода назад и на моделирование нагрузки

Добавление и удаление нагрузки с помощью Редактора Нагрузки

  • "Для добавления новой нагрузки на узел выберите "Редактор Нагрузки", затем выделите узел правой кнопкой мыши и нажмите кнопку добавить и заполните форму для добавления нагрузки.

  1. Выберите Редактор Нагрузки
  2. Выделите правой кнопкой мыши нужный узел и нажмите кнопу "Добавить"
  3. Введите название нагрузки и необходимые данные
  4. Нажмите кнопку "Добавить"
  • Для добавления новой нагрузки на канал связи выберите "Редактор Нагрузки", затем выделите канал связи правой кнопкой мыши и нажмите кнопку добавить и заполните форму для добавления нагрузки.

  1. Выберите Редактор Нагрузки
  2. Выделите правой кнопкой мыши нужный канал связи и нажмите кнопу "Добавить"
  3. Введите название нагрузки и необходимые данные
  4. Нажмите кнопку "Добавить"
